üîß Troubleshooting

‚ùå Common Issues & Solutions

1. "User not authenticated" error
  • Make sure you're logged in to your account
  • Check if your auth system is working properly
  • Try signing out and back in
2. "Unable to load conversations" error
  • Verify the database schema has been applied
  • Check your Supabase connection settings
  • Make sure RLS policies are enabled
3. Components not loading or showing errors
  • Run npm install to ensure all dependencies
  • Check browser console for TypeScript/JavaScript errors
  • Verify all imports are working correctly
4. Messages page is blank
  • Check if you have any profiles in your database
  • Try creating a test user and conversation
  • Look for errors in browser developer tools
